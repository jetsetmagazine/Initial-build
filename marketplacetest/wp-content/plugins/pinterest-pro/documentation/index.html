How To Install

Installing the plugin is very simple!

  1. FTP the pinterest-pro folder to your WordPress plugin directory ( /wp-content/plugins )
  2. Activate the plugin via the plugins menu in your WordPress dashboard.
  3. Now you are good to go!

Using The Shortcode Tool

To use the shortcode, you can use the easy to use TinyMCE plugin that is included!

Simply click the Pinterest icon that is now in your TinyMCE toolbar in the post editor.

Next, you will see tabs for both button types. Select the type of button you want to use and fill out the form.

When creating "Pin It" buttons, the description field is optional.

The fields are pretty straightforward, just enter your URLs and how you would like the buttons to appear then click insert. The shortcode will be automatically generated for you and inserted in the post! When the page / post is rendered your buttons will appear as normal.

Using The Template Tag

The template tag is just as easy to use!

Template tags work inside your themes as a simple function call. Keep in mind that if you use this function call, you need to make sure to leave the plugin enabled or it could cause errors. Why? Lets say you have the template tag called in your theme and then disable the plugin, that template tag function is no longer there to be found and thus it will error out.

Okay on to the tag!

There are a number of options you can specify in array format with the button. Below are all of the options as well as a few eamples of how to use the function.

type - You can specify either "follow" or "pinit" to choose what type of button you want to make.

follow_name - Only used for the follow button. This is the username you want the visitor to follow if they click the follow button.

follow_size - Only used for the follow button. This option is the size of the follow button you want to use, you can specify the follow values: small, pill, medium, and box.

pin_url - Only used with the pin it button. This option sets the URL you want to use for your pin it button.

pin_image_url - Only used with the pin it button. This is the url to the image you want pinned.

pin_counter - Enter how you would like the counter to be displayed on the pin it button. You can specify either "none", "horizontal", or "vertical"

pin_desc - An optional dsecription to use with your pin it button.

 

Template Tag Examples

Lets say you wanted to insert a pinterest follow button under every post on your site. You can simply edit your themes single.php file to include the template tag where you want the button to appear. It will then appear on every post!

<?PHP

// Display a follow button
tc_pinterest_pro( array("type" => "follow", "follow_name" => "username", "follow_size" => "medium") );

?>

As you can see using the template tag is very simple, just call the options you want to use in an array and in one line of code you can add buttons to every page on your site!

Here is an example using the pin it button:

<?PHP

// Create a Pin It button site wide
tc_pinterest_pro( array("type" => 'pinit', "pin_url" => 'http://tyler.tc/', "pin_image_url" => 'http://tyler.tc/portfolio/wp-content/uploads/2011/05/logo1.png', "pin_counter" => 'horizontal') );

?>

 

Using The Widgets

The widgets are very simple to use!

Just go to the widgets page as you normally would and you will see two new widgets. One for the follow button and one for the pin it button.

You can use them as many times as you want to create multiple buttons.

Simple fill out the settings just like the shortcode tool and click save!

Options Page

In the options page you can disable / enable the API for pinterest.

ONLY DO THIS if you have a theme or plugin that is not correctly loading the API. Sometimes loading things twice can cause errors so I built in a way to disable the loading of the API through this plugin incase you have a naughty plugin or theme on your site ;)
